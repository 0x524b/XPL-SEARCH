XPL SEARCH
===============
```
Search exploits in multiple exploit databases!
 Exploit databases available:
 * Exploit-DB
 * MIlw0rm
 * PacketStormSecurity
 * IntelligentExploit
 * IEDB
```


TO RUN THE SCRIPT
----
```
PHP Version       5.6.8
 php5-cli         Lib
cURL support      Enabled
 php5-curl        Lib
cURL Version      7.40.0
allow_url_fopen   On
Permission        Writing/Reading
```
